# Field

Monorepo root: Field UI work, Cursor project rules, and the Angular playground.

- **`field-demo/`** — Angular workspace (**components** library + **playground**). Run npm/CLI **from that folder**. Inside it: **`projects/components/`** (`field/`, `chips/`, `core/`), **`projects/playground/`** (hosted demo UI). Use `npm start` / `npm run build` / `npm run deploy:gh-pages` from `field-demo/`.
- **`.cursor/rules/`** — Cursor context for this workspace (open the **Field** folder in Cursor, not only `field-demo`).
