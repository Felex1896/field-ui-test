# Field

Monorepo root: Field UI work, Cursor project rules, and the Angular playground.

- **`field-demo/`** — Angular app (local dev, GitHub Pages build). Run npm/Angular CLI **from that folder** (`cd field-demo`, then `npm start`, `npm run build`, `npm run deploy:gh-pages`).
- **`.cursor/rules/`** — Cursor context for this workspace (open the **Field** folder in Cursor, not only `field-demo`).
